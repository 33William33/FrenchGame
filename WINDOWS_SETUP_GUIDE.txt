QUICK START (for beginners):
1. Double-click "start.bat"
2. Wait for setup to complete
3. Open browser and go to: http://localhost:3000

================================================================================
DETAILED SETUP:

STEP 1: Install Node.js
- Go to: https://nodejs.org/
- Click "Download for Windows" (choose LTS version)
- Run the downloaded installer (.msi file)
- Follow the installation wizard (use default settings)
- Restart your computer after installation

STEP 2: Run the Game
Option A - Easy way:
- Double-click "start.bat" in the game folder
- Wait for it to install dependencies and start

Option B - Manual way:
- Press Windows key + R
- Type "cmd" and press Enter
- Navigate to game folder: cd C:\path\to\your\game\folder
- Type: npm install
- Type: node app.js

STEP 3: Play the Game
- Open your web browser (Chrome, Firefox, Edge, etc.)
- Go to: http://localhost:3000
- Create an account by clicking "Create one"
- Enjoy the games!

TROUBLESHOOTING:

Problem: "node is not recognized"
Solution: Install Node.js from nodejs.org and restart computer

Problem: Port 3000 already in use
Solution: Close other programs that might be using port 3000

Problem: Permission denied errors
Solution: Right-click start.bat and "Run as administrator"

Problem: Browser shows "Can't reach this page"
Solution: Make sure the server is running (check command window)

================================================================================
STOPPING THE GAME:

To stop the server:
- Go to the black command window
- Press Ctrl + C
- Close the window
